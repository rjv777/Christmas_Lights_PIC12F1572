Кнопка key1 - переключает режимы.
Регулятор R6 - регулирует яркость и паузу.
Регулировка яркости происходит когда выбран режим статичный.
Регулировка паузы происходит в режиме мигания.

Key1 button - switches modes.
Knob R6 - Adjusts brightness and pause.
Brightness adjustment occurs when static mode is selected.
The pause is adjusted in flashing mode.

Q1-Q2-Q3 - KT815, KT817
